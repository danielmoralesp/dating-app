Getting Started
===============

Ottr is a modern dating site with old school romantic sensibilities

Ruby version - 2.2.3

Getting Started
===============

```bash
# Get the latest snapshot
git clone https://github.com/zkrzyzanowski/dating-app.git

# Install bundler gem
gem install bundler

# Install gem dependencies
bundle install

# Create database
rake db:migrate

# Start rails server
rails s
```
